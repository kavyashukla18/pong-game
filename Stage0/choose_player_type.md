# player choice

## Feature

    choose-player(): return single player or two player
    set-player-choice(): set the choice
  
## Scenario: player choose either he want to play with system

    Given: game is runing and connected to server
    When: player want to play the game with system
    Then: set the player choice

## Scenario: player choose either he want to play with another player

    Given: game is runing and connected to server
    When: player want to play the game with another player
    Then: set the player choice
